**🛍️ Store Launch Readiness Checklist**

**Branding & Design**
- [ ] Brand tokens (colors, fonts, spacing) applied consistently across all sections
- [ ] Logo, favicon, and brand imagery uploaded and crisp on all devices
- [ ] Typography scales consistent across headings, body, and CTAs
- [ ] Hero overlays readable without clashing with artwork
- [ ] Sticky header, search, and focus rings working correctly

**Products & Collections**
- [ ] All products have titles, descriptions, prices, and correct variants
- [ ] Product images are high quality, consistent size/ratio, and optimized
- [ ] Collections are organized and linked correctly in navigation
- [ ] Inventory levels set; out-of-stock behavior configured
- [ ] SKUs, barcodes, and metafields are clean and accurate

**Navigation & UX**
- [ ] Main menu and footer links all work (no broken links)
- [ ] Search returns relevant results
- [ ] Mobile responsiveness tested on iOS and Android
- [ ] Page load speed acceptable (test via PageSpeed Insights)
- [ ] 404 page is branded and helpful

**Checkout & Payments**
- [ ] Shopify Payments (or gateway) is live and tested with a real transaction
- [ ] Checkout shows correct currency (QAR)
- [ ] Shipping rates configured for all target zones
- [ ] Taxes set up correctly for Qatar
- [ ] Order confirmation email is branded and accurate

**Policies & Legal**
- [ ] Privacy Policy, Refund Policy, Terms of Service pages published
- [ ] Cookie consent banner active (if required)
- [ ] Contact page with accurate info live

**SEO & Metadata**
- [ ] Store title and meta description set
- [ ] All product/collection pages have unique SEO titles and descriptions
- [ ] Image alt text filled in
- [ ] Sitemap submitted to Google Search Console
- [ ] No PII leaking in URLs or metadata

**Apps & Integrations**
- [ ] All installed apps tested and conflict-free
- [ ] Email/SMS (Klaviyo or active app) flows set up — welcome, abandoned cart, order confirmation
- [ ] Analytics (Google Analytics / Shopify Analytics) tracking verified

**Theme QA**
- [ ] Tested on a duplicate/dev theme before publishing
- [ ] All custom Liquid/CSS/JS reviewed and no console errors
- [ ] GitHub sync up to date with final theme state
- [ ] Rollback plan documented

**Final Go-Live**
- [ ] Custom domain connected and SSL active
- [ ] Password page removed
- [ ] Test order placed and fulfilled end-to-end
- [ ] Store announced (social, email, etc.)
