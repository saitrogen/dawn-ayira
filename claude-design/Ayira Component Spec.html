Button

The primary interactive control for user actions. Four visual variants cover every commerce context. Uses Hanken Grotesk weight 600, Title Case for commerce actions, sentence case for navigational labels. Scale-press on active state.

All variants — default size

Block (full-width) · used in cart footer
Variants
Primary Forest-700 bg · Cream-50 text · no border. The one main action per view.
Secondary Cream-100 bg · Forest-700 text · 1.5px Forest-100 border. Paired with Primary.
Accent Mustard-500 bg · Forest-900 text · no border. Hero & promo CTAs only.
GhostTransparent bg · Forest-700 text. Low-emphasis or paired with Accent on dark bg.
Sizing

Large17px · weight 600 · padding 16px 28px · radius 14px

One Primary button per section. Pair Accent + Ghost on forest backgrounds (hero, promo strip).
Don't

04

Logo

A two-part identity: a wordmark ("ayira Minimart" in a custom high-contrast modern serif lockup) and a circular brandmark (sun emblem). Both are image files — never re-typeset. The serif is exclusive to the logo; all UI uses Hanken Grotesk.
