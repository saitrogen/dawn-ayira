How to apply these in the Dawn theme customizer

SchemeDawn section · settingWhy
01 · Cream color_scheme: Background 1 Default for every section that doesn't explicitly override. Home page sections, collection grid, PDP body, cart page.
02 · White color_scheme: Background 2 Cards, drawers, predictive-search dropdown. Use for elevated surfaces only — never as a page background.
03 · Forest color_scheme: Inverse Header (sticky), Announcement bar, Footer, Image banner overlay. The dark scheme Dawn uses for "Inverse" by default.
04 · Dark Ink color_scheme: Accent 2 Optional alternate dark footer. Use only when you specifically want cooler black; forest is the primary inverse.
05 · Mustard color_scheme: Accent 1 Apply to a single Image-banner or Rich-text section during promo windows. Designed to be intentionally loud — keep it short-lived.
